## Key Pitfall: buildFuture

Hugo v0.163+ **silently skips** posts with dates in the future. Use either:
- `buildFuture = true` in `hugo.toml` (persistent)
- `hugo --buildFuture` flag in the build command

Diagnose with:
```bash
hugo list all                  # See all pages including future-dated
find public -name '*.html'     # Check which files were actually generated
```
