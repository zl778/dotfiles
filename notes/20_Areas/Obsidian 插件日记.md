# Heading Shifter (标题层级调整器） 2026-06-01

**Heading Shifter** 是一款用于快速调整 Markdown 标题层级的 Obsidian 插件。它允许用户通过快捷键或命令，一键提升或降低当前标题及其子标题的层级，无需手动修改多个 `#` 符号。

该插件特别适合整理长篇笔记、重构知识体系、调整文档大纲结构等场景。通过使用 **Tab 和 Shift+Tab（可配置）** 即可快速完成标题层级转换，大幅提升 Markdown 大纲编辑效率。

**主要功能：**

- 提升标题层级（如 H3 → H2）
- 降低标题层级（如 H2 → H3）
- 支持批量调整选中标题及其子级结构
- 支持自定义快捷键
- 与 Obsidian 大纲式笔记工作流深度结合

**适用场景：**

- 重构笔记目录结构
- 调整文章大纲层级
- 整理项目文档
- 构建和维护知识库体系

对于经常使用 Markdown 标题组织内容的用户来说，Heading Shifter 能显著减少手动编辑标题符号的工作量，让大纲管理更加高效。
