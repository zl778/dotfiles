### 常用操作

| 操作         | 快捷键                                         | 说明              |
| ---------- | ------------------------------------------- | --------------- |
| **全屏截图**   | `⇧ Shift + ⌘ Command + 3`                   | 在光标所在的 Mac 上截图  |
| **区域截图**   | `⇧ Shift + ⌘ Command + 4`                   | 同上              |
| **截图工具面板** | `⇧ Shift + ⌘ Command + 5`                   | 调出截图控制面板        |
| **页面翻页**   | `Fn + ↑/↓` 或 `空格/Shift+空格`                  | 取决于应用，直接在当前设备生效 |
| **断开所有连接** | `⌃ Control + ⌥ Option + ⌘ Command + Delete` | 快速禁用通用控制所有设备连接  |
|            |                                             |                 |
1. **快捷键跟随光标**：例如光标在 Mac A 时按 `⌘ Command + C` 会复制 Mac A 的内容；将光标移到 Mac B 后，同一快捷键会操作 Mac B
2. **应用切换**：`⌘ Command + Tab` 仅在当前设备内切换应用，**不会跨设备切换**

### 一、通用控制下的触摸板工作方式

| 场景                  | 触摸板行为                      |     |
| ------------------- | -------------------------- | --- |
| **光标在 Mac A**       | 所有手势（三指滑动、四指缩放等）仅作用于 Mac A |     |
| **光标推到边缘切换到 Mac B** | 触摸板自动"移交"控制权，后续手势作用于 Mac B |     |
| **跨设备拖拽文件**         | 双指按住文件拖到屏幕边缘，松开后自动传输到另一台设备 |     |
|                     |                            |     |

✅ **支持的手势**（在当前设备上正常工作）：
- 三指上滑 → 调度中心（Mission Control）
- 三指左右滑 → 切换桌面空间（Spaces）
- 四指向内捏 → 启动台（Launchpad）
- 双指滚动 → 页面滚动
