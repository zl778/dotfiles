#a/学习/软件

视频讲述了 Obsidian 的实用进阶用法，涵盖免费云同步、AI 结合、图片存储、手机端使用、格式导出等多个方面，且所有工具均免费，具体如下：

- **选择 Obsidian 的三大核心原因**：一是数据安全，笔记为本地独立 markdown 文件，即使软件开发者跑路也可继续使用，还能通过 GitHub 备份；二是界面丝滑流畅，操作无卡顿，不破坏工作心流；三是与 AI 工具适配度高，可实现 AI 查笔记、整理文件夹、模仿文风写作等多种功能。

- **GitHub 实现 Obsidian 免费云同步**：先注册登录 GitHub，创建私有存储库，通过 GitHub Desktop 克隆到本地，将 Obsidian 关联该本地文件夹；新建.gitignore 文件排除易冲突的 workspace.json；安装 Obsidian 的 Git 插件，设置停止编辑 1 分钟后自动提交、启动软件时拉取远端改动，实现自动化同步，也可结合网盘、硬盘做冷备份。

- **Obsidian 接入 AI 的两种方式及实操玩法**：方式一是用社区 AI 插件，方式二是用大厂 AI 编程工具（推荐），以免费的 Gemini CLI 为例，需先安装 node.js，再在 Obsidian 笔记目录终端安装并配置 Gemini CLI；可让 AI 根据往期内容生成视频选题、批量整理文件夹、模仿文风写视频脚本，且借助 Git 可还原 AI 的文件修改，无需担心文件损坏丢失；若需 AI 搜索文件、搭建本地知识库，可搭配工具 hyper link。

- **Obsidian 的 Markdown 基础语法**：介绍了标题、加粗、删除、高亮、代码块、引用、有序 / 无序列表的基础语法，表格、分隔线、公式块可通过右键插入，掌握这些能使用 Obsidian80% 的功能。

- **插件优化 Obsidian 图片存储方案**：Obsidian 原生图片存储会让目录杂乱，且链接非标准 markdown 语法；安装 Custom attach location 插件，配置 markdown url 格式、重命名模式等，再在系统设置中调整文件与链接相关选项，实现图片按规范存储，链接符合标准语法，可在其他软件正常渲染，还能直接调整图片显示大小，该方案为本地存储，安全隐私且无费用。

- **安卓手机端使用 Obsidian 并实现云同步**：将电脑端 Obsidian 笔记文件夹复制到手机指定目录，在手机端 Obsidian 打开该文件夹完成基础配置；在手机端 Git 设置中填写 GitHub 用户名、邮箱，在电脑端生成并复制 GitHub 的 personal access token 粘贴至手机，完成同步配置；手机端编辑的笔记可自动同步到 GitHub 和电脑端，注意避免手机与电脑同时编辑同一文件，冲突需手动解决。

- **Obsidian 笔记导出多格式的方法**：安装 Enhancing export 插件，下载并配置 pandoc 文件格式转换工具的路径，右键选中笔记即可导出为 Word、HTML 等多种格式，导出的笔记配图可正常显示。

- **Obsidian 双向链接与知识图谱的使用**：通过两对方括号创建笔记间的双向链接，可修改链接名称，按住 CTRL 键可打开链接笔记；点击查看关系图谱，能以图形化形式展示笔记间的关联，助力知识归纳整理，发现知识间隐藏关联并激发灵感。
