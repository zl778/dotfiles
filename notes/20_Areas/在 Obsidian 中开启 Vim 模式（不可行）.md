#a/学习/软件

在 Obsidian 中开启 **Vim 模式**（设置 → 编辑器 → Vim 模式）后，编辑器会模拟 Vim 的操作逻辑，但**并非完整 Vim 实现**（基于 CodeMirror 的简化版）。以下是 **2024 年最新版 Obsidian (1.7.0+)** 中 Vim 模式的**核心操作技巧**，按使用场景分类整理：

---

### ⚙️ **基础设置（必看）**
| 操作            | 说明                                            |
| ------------- | --------------------------------------------- |
| **开启 Vim 模式** | 设置 → 编辑器 → 开启 `Vim 模式` → **重启 Obsidian**（关键！） |
| **默认状态**      | 普通模式（Normal Mode）→ 用 `i`/`a` 进入插入模式           |
| **退出 Vim 模式** | 按 `Esc` 退出所有模式 → 按 `Ctrl + ,` 临时禁用（调试用）       |

> 💡 **关键区别**：  
> - **无真正的命令行模式**（如 `:w` 无效）  
> - **不支持 `.vimrc`**，需在 Obsidian 设置中配置  
> - **`<Leader>` 键默认为 `\`**（可自定义：设置 → Vim 模式 → `<Leader>` 键）

---

### 🚀 **核心操作技巧速查表**
#### ✅ **1. 模式切换（最常用）**
| 按键    | 作用      | 说明            |
| ----- | ------- | ------------- |
| `i`   | 进入插入模式  | 光标后插入         |
| `a`   | 进入插入模式  | 光标**后**插入（常用） |
| `I`   | 进入插入模式  | 行首插入          |
| `A`   | 进入插入模式  | 行尾插入          |
| `o`   | 新建行并插入  | 下方新建行         |
| `O`   | 新建行并插入  | 上方新建行         |
| `Esc` | 退出插入模式  | 返回普通模式        |
| `v`   | 进入可视模式  | 选中文本          |
| `V`   | 进入行可视模式 | 选中整行          |

#### ✅ **2. 光标移动（高效替代鼠标）**
| 按键              | 作用         | 说明                 |
| --------------- | ---------- | ------------------ |
| `h`/`j`/`k`/`l` | 左/下/上/右    | 基础移动               |
| `w`             | 跳转到下一个单词开头 | `2w` = 跳 2 个单词     |
| `b`             | 跳转到上一个单词开头 | `3b` = 跳 3 个单词     |
| `0`             | 行首         | `^` = 行首非空格        |
| `$`             | 行尾         | `g$` = 行尾（含空格）     |
| `gg`            | 跳转到文档顶部    | `G` = 跳转到文档底部      |
| `Ctrl + d`      | 向下翻半屏      | `Ctrl + u` = 向上翻半屏 |
| `f` + `字符`      | 跳转到行内字符    | 如 `f.` 跳到句号        |

#### ✅ **3. 文本编辑（删除/复制/粘贴）**
| 按键   | 作用      | 说明              |
| ---- | ------- | --------------- |
| `x`  | 删除当前字符  | `2x` = 删除 2 个字符 |
| `dw` | 删除到单词结尾 | `d$` = 删除到行尾    |
| `dd` | 删除整行    | `2dd` = 删除 2 行  |
| `yy` | 复制当前行   | `2yy` = 复制 2 行  |
| `p`  | 粘贴      | `P` = 粘贴到光标前    |
| `u`  | 撤销      | `Ctrl + r` = 重做 |

#### ✅ **4. Obsidian 特有功能（Vim 模式专属）**
| 按键               | 作用           | 说明                      |
| ---------------- | ------------ | ----------------------- |
| `<Leader>` + `f` | 跳转到文件        | 默认 `\f`（需先按 `<Leader>`） |
| `<Leader>` + `t` | 打开命令面板       | 默认 `\t`                 |
| `<Leader>` + `v` | 打开视图切换       | 默认 `\v`                 |
| `g` + `g`        | 跳转到文档顶部      | 与 `gg` 相同               |
| `g` + `l`        | 跳转到文档底部      | 与 `G` 相同                |
| `]` + `[`        | 跳转到下一个/上一个标题 | 适合长文档导航                 |

> 💡 **`<Leader>` 键说明**：  
> - 默认为 `\`（需按 `Shift + \` 输入 `\`）  
> - 可修改为 `Ctrl`（设置 → Vim 模式 → `<Leader>` 键 = `Ctrl`）  
> - 例如：`Ctrl + f` = 跳转到文件

---

### ⚠️ **重要限制（避坑指南）**
| 问题 | 解决方案 |
|------|----------|
| **`:w` 保存无效** | Obsidian 无命令行模式 → 用 `Ctrl + S` 保存 |
| **搜索冲突** | 普通模式下按 `/` 会触发 Obsidian 搜索 → 用 `Ctrl + F` 代替 |
| **粘贴板问题** | `p` 粘贴的是 Vim 内部寄存器 → 用 `Ctrl + V` 粘贴系统剪贴板 |
| **无法跳转外部链接** | `gf` 无效 → 用 `Ctrl + Click` 或 `<Leader> + f` |
| **插件兼容性差** | 部分插件（如 Dataview）在 Vim 模式下异常 → 临时禁用 Vim |

---

### 🔧 **高级配置（推荐）**
1. **修改 `<Leader>` 键**（避免按 `\` 麻烦）  
   ```yaml
   # 设置 → Vim 模式 → <Leader> 键 → 选择 "Ctrl"
   ```
2. **启用可视模式行选择**  
   ```yaml
   # 设置 → Vim 模式 → 可视模式 → 勾选 "行选择"
   ```
3. **自定义快捷键**  
   ```yaml
   # 在 "Vim 模式" 设置中 → 添加自定义键位映射
   map <Leader>t :command Open command palette<CR>
   ```

---

### 📌 **一句话总结**
> **Vim 模式 = 基础 Vim 逻辑 + Obsidian 专属扩展**  
> - **日常编辑**：用 `hjkl` 移动 + `i/a` 编辑 + `dw/dd` 删除  
> - **高效导航**：`<Leader> + f` 跳文件 + `] [` 跳标题  
> - **避坑重点**：**无 `:w` 命令**，用 `Ctrl + S` 保存；**搜索用 `Ctrl + F`**

---

### 💡 终极建议
1. **先关闭 Vim 模式** → 用 `Ctrl + ,` 打开 **命令面板** → 搜索 `Vim Mode`  
2. **开启后立即测试**：  
   ```markdown
   # 测试 Vim 模式
   普通模式下按 `i` → 输入文字 → 按 `Esc` → 按 `dw` 删除单词
   ```
3. **遇到问题**：按 `Esc` 退出 → 用 `Ctrl + ,` 临时禁用 Vim 模式

> 🌟 **推荐学习路径**：  
> **基础移动（hjkl）→ 模式切换（i/a/esc）→ 文本删除（dw/dd）→ Obsidian 专属命令（<Leader>）**  
> （Obsidian Vim 模式**不适合 Vim 老手**，但对**新手提升编辑效率极佳**）
