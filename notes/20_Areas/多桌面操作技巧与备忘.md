## macOS 中的多桌面（Spaces）
### 一、切换桌面

#### 三指或四指左右滑动

设置位置：
**系统设置 → 触控板 → 更多手势 → 在全屏应用程序之间轻扫**


---

### 二、查看所有桌面

#### 三指或四指向上轻扫

显示当前所有桌面和窗口。

  
---
### 三、新建桌面

#### 三指或四指向上轻扫 点击右上角 **+**

新增一个桌面。

---
### 四、移动窗口到其他桌面

#### 方法1：三指或四指向上轻扫 直接把窗口拖到顶部目标桌面。
#### 方法2：点击窗口 OP+1，2，3

#### 方法3：右点击DOCK图标-〉选项-〉指定到桌面

适合微信、企业微信、飞书等常驻应用。

---

### 五、直接跳转到指定桌面

macOS 默认没有开启。

可以自己设置：

**系统设置 → 键盘 → 键盘快捷键 → Mission Control**

里面可以设置：

- 切换到桌面1
    
- 切换到桌面2
    
- 切换到桌面3
   
例如：

| 功能  | 快捷键         |     |
| --- | ----------- | --- |
| 桌面1 | OP+ 1       |     |
| 桌面2 | OP+ 2       |     |
| 桌面3 | OP + 3      |     |

配置后效率极高。

---

### 六、全屏应用切换

当应用进入全屏后，会自动变成独立桌面。

```text
三指左右滑动
```

例如：

```text
桌面1：Chrome
桌面2：微信
桌面3：Cursor
桌面4：终端
桌面5：全屏 Obsidian
桌面6：全屏 Claude
```

直接左右切换即可。

## Win11 下的多桌面（Spaces）

### Win+tab 调出多窗口界面 鼠标操作即可
备用快捷：（没啥用）
Win + Ctrl + ←/→     左/右切换桌面
Win + Ctrl + F4        关闭桌面
