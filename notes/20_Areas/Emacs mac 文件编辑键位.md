#a/学习/系统

## 移动光标
 - Ctrl + A：到行首（你已经在用）。StartOfLine
 - Ctrl + E：到行尾（你已经在用）。EndOfLine
 - Ctrl + F：向右移动一个字符（相当于右方向键）。CharRight
 - Ctrl + B：向左移动一个字符（相当于左方向键）。CharLeft
 - Ctrl + N：向下一行。LineDown
 - Ctrl + P：向上一行。LineUp
   
（配合方向键的那套：Command+←/→ 到行首/行尾，Command+↑/↓ 到文档首/尾；Option+←/→ 以「词」为单位移动。）

## 删除与编辑
- Ctrl + D：删除光标后的一个字符（类似 Delete）。
- Ctrl + H：删除光标前的一个字符（类似退格 Backspace）。
- Ctrl + K：从光标删到行尾。（很多人拿它当「清空到行尾」用。）
- Ctrl + U：从光标删到行首（部分场景等价于 Command + Delete）。DeleteBackWord
- Ctrl + T：交换光标前后两个字符（打错两个字母时很好用）。
    
## 选中整段/整篇

- Shift + Command + ←：从光标选到行首。    
- Shift + Command + →：从光标选到行尾。 
- Shift + Command + ↑：从光标选到文稿开头。
- Shift + Command + ↓：从光标选到文稿末尾。
- Shift + Option + ← / →：按「词」扩展选择范围（左边/右边一词一词选）。    

## 常用系统级快捷键（顺带一提）

- Command + C / X / V：复制 / 剪切 / 粘贴。    
- Command + Z / Shift + Command + Z：撤销 / 重做。
- Command + 左箭头 / 右箭头：行首 / 行尾（和 Ctrl+A/E 功能类似但更「Mac 风格」）。   

一个简单的记法：

- 以 Ctrl 打头的是「细粒度移动/编辑」（字符、行）。
- 加上 Command / Option / Shift 就变成「大范围移动/选中」（整行、整文档、按词）。
