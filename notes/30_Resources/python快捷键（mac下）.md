### **🔥 核心 Python 开发快捷键**
#### **1. 代码编辑**

| 快捷键                    | 功能说明               |
| ---------------------- | ------------------ |
| `Cmd + /`              | 快速注释/取消注释当前行或选中代码块 |
| `Option + ↑/↓`         | 向上/向下移动当前行         |
| `Shift + Option + ↑/↓` | 复制当前行到上/下方         |
| `Cmd + D`              | 选中当前单词（重复按可多选相同词）  |
| `Cmd + Shift + L`      | 选中所有与当前选中内容相同的文本   |
#### **2. 代码导航**

| 快捷键                 | 功能说明          |
| ------------------- | ------------- |
| `Cmd + P`           | 快速跳转到文件       |
| `Cmd + T`           | 全局符号搜索（类/函数名） |
| `F12` 或 `Cmd+Click` | 跳转到定义         |
| `Option + F12`      | 预览定义（不跳转）     |
| `Ctrl + -`          | 返回上一个光标位置     |

#### **3. 代码执行与调试**

| 快捷键             | 功能说明             |
| --------------- | ---------------- |
| `Shift + Enter` | 在交互式窗口运行当前行/选中代码 |
| `F5`            | 启动调试             |
| `Ctrl + F5`     | 运行当前文件（不调试）      |
| `F9`            | 切换断点             |
| `F10`           | 单步跳过（调试时）        |
| `F11`           | 单步进入（调试时）        |

#### **4. 代码格式化**

| 快捷键                  | 功能说明     |
| -------------------- | -------- |
| `Shift + Option + F` | 格式化整个文档  |
| `Cmd + K → Cmd + F`  | 格式化选中代码块 |

---

### **🎯 Python 专属增强操作**

1. **快速触发智能建议** • 输入时自动触发，或手动按 `Ctrl + Space`（需关闭系统输入法快捷键冲突）
    
2. **查看函数文档** • 悬停显示：鼠标悬停在函数上 • 快捷键：`Cmd + K → Cmd + I`（显示悬浮提示）
    
3. **重构代码** • 重命名符号：`F2` • 提取变量：选中表达式 → `Cmd + .` → 选择 "Extract variable"

| ` Ctrl + `` `     | 快速切换终端        |

### **💡 效率技巧**

1. **快速修复错误** 遇到红色波浪线时按 `Cmd + .` 可直接显示修复建议（如自动导入模块）
    
2. **批量操作** • 多光标编辑：按住 `Option` 点击不同位置 • 列选择：按住 `Option` 拖动鼠标
    
3. **Jupyter Cell 操作**（适用于 `.ipynb` 文件） • `Shift + Enter`：运行当前 Cell • `Option + Enter`：运行当前 Cell 并插入新 Cell

### **⚠️ 注意事项**

• 如果快捷键失效，检查是否与其他软件冲突（如输入法、Alfred 等） • 可通过 `Cmd + K → Cmd + S` 自定义快捷键
