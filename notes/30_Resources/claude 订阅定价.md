### 什么是 Claude Code？

Claude Code 是 Anthropic 推出的**终端命令行编码代理工具**（agentic coding tool），它：
- 以 CLI 形式运行在你的本地终端中
- 通过 API 连接到 Anthropic 的 Claude 服务 
- 能理解代码库、编辑文件、执行命令，帮助开发者更快编码

⚠️ **关键点**：Claude Code 本身**不单独收费**，它的使用消耗计入你现有的 **Claude 订阅额度**或 **API 调用费用**。

### ⚠️ 警惕第三方"共享账号"服务

部分中文网站（如 claudecode-cn.com）宣传的 "$259/月 Claude Code 订阅" 实为：
- **非官方的共享账号服务**（多人共用一个 Anthropic 账号）
- 存在账号安全风险和稳定性问题 
- **并非 Anthropic 官方产品或定价**

✅ **正确理解**：Claude Code 是工具，不是订阅产品
