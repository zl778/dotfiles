# PKM 项目上下文

## 项目概述

这是一个基于 **Obsidian** 的个人知识管理（PKM - Personal Knowledge Management）库，使用 iCloud 同步在 macOS、iOS 等多平台之间共享知识库。

### 核心特点

- **双端同步**：通过 iCloud 文档在 Mac 和 iOS 设备间无缝同步
- **PARA 系统**：采用 Projects（项目）、Areas（领域）、Resources（资源）、Archives（存档）的笔记组织方法
- **AI 辅助**：集成了 Obsidian Copilot 插件，提供智能对话、网页抓取、文本处理等功能
- **模板化工作流**：使用 Templater 插件实现高效的笔记创建流程

### 主要技术栈

- **笔记系统**：Obsidian（Markdown）
- **云同步**：iCloud Drive
- **AI 集成**：Obsidian Copilot
- **模板引擎**：Templater Obsidian
- **绘图工具**：Excalidraw
- **数据可视化**：Maps、Dataview

## 使用说明

### 创建新笔记

1. **普通笔记**：新笔记默认创建在 `pages/` 文件夹
2. **日记**：通过 Calendar 插件创建，存储在 `journals/` 文件夹，格式为 `YYYY_MM_DD.md`
3. **使用模板**：从 Template 文件夹选择模板

### Copilot AI 功能

Copilot 提供以下自定义提示词：
- Clip Web Page - 网页抓取
- Clip YouTube Transcript - YouTube 字幕抓取
- Explain like I am 5 - 简单解释
- Fix grammar and spelling - 语法拼写修正
- Generate glossary - 生成词汇表
- Generate table of contents - 生成目录
- Make longer/shorter - 文本扩展/精简
- Simplify - 简化文本
- Summarize - 摘要
- Translate to Chinese - 翻译成中文

### Templater 常用语句

```javascript
// 文件相关
<% tp.file.title %>              // 文件标题
<% tp.file.creation_date() %>    // 创建日期

// 日期相关
<% tp.date.now() %>              // 当前日期
<% tp.date.tomorrow() %>         // 明天
<% tp.date.yesterday() %>        // 昨天

// 系统交互
<% tp.system.prompt("输入作者") %>  // 用户输入
<% tp.system.suggester([...]) %>   // 选项选择
```

### 附件管理

- 所有附件（图片、PDF、Excel 等）自动保存到 `assets/` 文件夹
- Excalidraw 绘图保存在 `Excalidraw/` 文件夹

## 笔记类型

### 主要分类（pages/）

- **技术学习**：Python、Docker、Git、Emacs、Vim 等工具使用
- **办公软件**：Excel、Word 宏函数、公式技巧
- **工程相关**：建筑、机电、智能化工程标书、技术文档

- **AI 工具**：Claude、DeepSeek、Gemini、Kimi 等使用心得

- 技术笔记
- 项目总结

### 周记

- 格式：`YYYY-Wxx.md`（如 2026-W07.md）
- 用于周度总结和规划

## PARA 系统说明

- **Projects（项目）**：有明确目标和截止日期的任务集合
- **Areas（领域）**：需要长期维护标准的活动领域
- **Resources（资源）**：持续感兴趣的主题和参考资料
- **Archives（存档）**：来自上述三类的非活跃条目

## 注意事项

1. **iCloud 同步**：确保在所有设备上正确配置 iCloud Drive
2. **附件路径**：附件始终保存在 `assets/` 文件夹
3. **回收站**：本地删除的文件进入 `.trash/` 文件夹
4. **Copilot 对话**：AI 对话记录保存在 `copilot/copilot-conversations/`
5. **模板文件**：修改模板时注意备份

## 文件命名约定

- **日记**：`YYYY_MM_DD.md`
- **周记**：`YYYY-Wxx.md`
- **Excalidraw**：`Drawing YYYY-MM-DD HH.MM.SS.excalidraw.md`
- **其他笔记**：使用清晰的中文标题

## 扩展功能

- **地图视图**：使用 Maps 插件创建基于地理位置的笔记
- **数据查询**：使用 Dataview 插件进行笔记数据查询
- **绘图**：使用 Excalidraw 创建流程图、思维导图等
- **终端**：可直接在 Obsidian 中运行命令行操作
