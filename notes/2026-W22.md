可以。下面这份按 **Ghostty 终端 + macOS + 默认 zsh/emacs 模式** 来整理，日常够用了。 快捷键 快捷操作

**移动光标**

| 操作       | 快捷键                |
| -------- | ------------------ |
| 跳到行首     | `Ctrl + A、COM　＋　←` |
| 跳到行尾     | `Ctrl + E、COM　＋　→` |
| 向左移动一个字符 | `←、Ctrl + B`       |
| 向右移动一个字符 | `→、Ctrl + F`       |
| 向左移动一个词  | `Option + ←`       |
| 向右移动一个词  | `Option + →`       |
| 向上移动一行   | `Ctrl + P`         |
| 向下移动一行   | `Ctrl + N`         |

**删除内容**

| 操作        | 快捷键                                       |
| --------- | ----------------------------------------- |
| 删除光标前一个字符 | `Ctrl + H、Backspace`                      |
| 删除光标后一个字符 | `Ctrl + D`                                |
| 剪切光标前一个词  | `Option + Backspace`                      |
| 剪切光标后一个词  | `Option + D`                              |
| 剪切光标到行首   | `Ctrl + U、COM＋Backspace、Option＋Backspace` |
| 剪切光标到行尾   | `Ctrl + K`                                |
| 剪切整行      | `Ctrl + U`，如果光标在行尾                        |
|           | （可用`Ctrl + Y`粘贴回来）                        |

**撤销与恢复**

| 操作        | 快捷键                      |
| --------- | ------------------------ |
| 撤销刚才的编辑   | `Ctrl + _、Ctrl + ?`      |
|           |                          |

`Ctrl + _` 是下划线，通常等于按住 `Ctrl + Shift + -`。

**历史命令**

| 操作             | 快捷键                |
| -------------- | ------------------ |
| 上一条命令          | `↑` 或 `Ctrl + P`   |
| 下一条命令          | `↓` 或 `Ctrl + N`   |
| 搜索历史命令         | `Ctrl + R`         |
| 退出历史搜索         | `Esc` 或 `Ctrl + G` |
| 重新执行上一条命令      | `!!`               |
| 替换上一条命令中的文字并执行 | `^旧内容^新内容^`        |

例子：

```bash
^-language^--language zh^
```

**复制粘贴**

| 操作 | 快捷键 |
|---|---|
| 复制选中文本 | `Command + C` |
| 粘贴 | `Command + V` |
| 终端传统粘贴 | `Ctrl + Y` |
| 剪切光标到行尾后粘贴回来 | `Ctrl + K` 后 `Ctrl + Y` |
| 剪切光标到行首后粘贴回来 | `Ctrl + U` 后 `Ctrl + Y` |

注意：`Ctrl + K`、`Ctrl + U` 删除的内容会进入终端自己的剪切缓冲区，可以用 `Ctrl + Y` 粘回来。

**常用控制**

| 操作          | 快捷键        |
| ----------- | ---------- |
| 中断当前运行命令    | `Ctrl + C` |
| 暂停当前输出/程序   | `Ctrl + Z` |
| 清屏          | `Ctrl + L` |
| 结束输入/退出某些程序 | `Ctrl + D` |
| 停止屏幕输出      | `Ctrl + S` |
| 恢复屏幕输出      | `Ctrl + Q` |

如果你不小心按了 `Ctrl + S`，终端像卡住一样，按 `Ctrl + Q` 就恢复。

**补全与修改**

| 操作              | 快捷键                       |
| --------------- | ------------------------- |
| 自动补全文件名/命令      | `Tab`                     |
| 显示更多补全候选        | 连按两次 `Tab`                |
| 交换光标前两个字符       | `Ctrl + T`                |
| 把当前行暂存，先输入另一条命令 | `Ctrl + Q` 不建议用，容易和恢复输出混淆 |

**最常用组合**

改上一条命令开头：

```text
↑
Ctrl + A
编辑
Enter
```

删除光标后所有内容：

```text
Ctrl + K
```

删除光标前所有内容：

```text
Ctrl + U
```

搜索以前敲过的命令：

```text
Ctrl + R
输入关键词
Enter
```

如果 `Option + ←/→` 或 `Option + D` 不生效，通常是 Ghostty 里 Option 没被当作 Meta/Alt。可以先用 `Ctrl + A/E/K/U/R` 这些，最稳定。

在～/.zshrc 增加一行
	bindkey '^U' backward-kill-line
重新加载 确认已调整
	bindkey | grep '\^U'
完工

**在 macOS 文本编辑里常用方式是：**
- Cmd + Delete：删除光标前面到本行开头的内容。
- Option + Delete：删除光标前面的一个单词
- Ctrl + H：相当于 Backspace，删除光标前一个字符
- Ctrl + K：删除光标后面到行尾
