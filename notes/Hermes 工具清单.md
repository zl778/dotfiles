## 控制命令

```bash
hermes tools                    # 交互式启用/禁用
hermes tools list               # 列出所有工具及状态
hermes tools enable <名称>      # 启用工具
hermes tools disable <名称>     # 禁用工具
```

> 注：更改工具需 `/reset` 新会话才能生效。
